# מה אוכלים? 🍽️

אתר אישי שעוקב אחרי המלצות האוכל של [TimeOut ישראל](https://timeout.co.il/topic/אוכלים-שותים/)
ומציג אותן ככרטיסים — כרטיס לכל מסעדה/בית קפה/בר, גם כשכתבה אחת ממליצה על כמה מקומות.

## איך זה עובד
- `scraper.py` — רץ אוטומטית פעם ביום (GitHub Actions), מושך כתבות חדשות מעמוד
  הנושא "אוכלים ושותים", מפצל כל כתבה לכרטיסים ומחלץ פרטים בעזרת חוקים ולקסיקונים.
- `docs/data.json` — כל הכרטיסים שנאספו (רק כתבות חדשות מתווספות — יש זיהוי כפילויות).
- `docs/index.html` — האתר עצמו (עברית, RTL, מצב כהה, חיפוש וסינון), מוגש ב-GitHub Pages.

## הערה
לשימוש אישי בלבד. הזכויות על התוכן וההמלצות שייכות ל-TimeOut ישראל,
וכל כרטיס מקשר לכתבה המקורית. הסריקה מוגבלת לפעם ביום, בנימוס.
